üß† What is an Operating System (OS)?

An OS is system software that manages hardware & software resources, and provides an interface between the user and computer hardware.

Main Functions of OS:

  1. Hardware Access (e.g., keyboard, disk, screen)
  2. User Interface (CLI, GUI)
  3. Resource Management (memory, CPU, devices)
  4. Abstraction ‚Äì hides complex hardware details
  5. Program Execution ‚Äì runs apps safely & efficiently

üö´ What If There's No OS?

  • Every application would need to manage hardware on its own ‚Äî making it bulky and complex.
  • No resource sharing ‚Üí Only one app may dominate.
  • No memory protection ‚Üí One app could corrupt others.

üß± What is an OS Made Up Of?

A collection of system software that works together to manage and operate the system.

üéØ Goals of an Operating System

  • Maximize CPU utilization

  • Prioritize important jobs
  • Ensure smooth execution of multiple apps

üîÑ Key Concepts

1. Batch Processing OS

User submits job using punch cards ‚Üí Operator groups & submits them.

Disadvantages:

  • No priority support ‚Üí high-priority jobs may starve.
  • CPU waits during I/O ‚Üí less efficient.

2. Multiprogramming OS

  • Multiple jobs in memory; when one waits (I/O), another runs.
  • Single CPU, but efficient ‚Üí less CPU idle time.
  • Context switching occurs.

3. Multitasking OS

  • Extension of multiprogramming.
  • Multiple apps/tasks handled seemingly at the same time.
  • Based on time-sharing.
  • Example: Editing a doc while listening to music.

4. Multiprocessing OS

  • Multiple CPUs.
  • Improved throughput, fault-tolerance, and parallelism.
  • Example: Servers, high-performance computing.

5. Distributed OS

  • Resources like CPUs, memory, and storage are distributed over a network.
  • Each node acts independently but works as a single system.
  • Example: Google File System, cloud platforms.

6. Real-Time OS (RTOS)

  • Used for systems that require immediate response.
  • Time-critical applications.
  • Examples: Air traffic systems, robots, medical devices.
